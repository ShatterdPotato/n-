# n#
Don't let the name of the repository fool you, this game is called n#, NOT n-

## What is n#?
n# is an open-source fanmade recreation of the 2D platformer series [n](https://www.thewayoftheninja.org/), [n+](https://www.xbox.com/en-US/games/store/n/bsp47tqwf6wd), and [n++](https://store.steampowered.com/app/230270/N_NPLUSPLUS/).
